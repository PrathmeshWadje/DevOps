docker network create       -   create docker internal network (specify the driver)
docker network ls           -   see all the networkds
docker instpect {id | name} -   check details of the network
docker run --name {container name} --network=none {container name}


// example task:
// Create a new network named wp-mysql-network using the bridge driver. Allocate subnet 182.18.0.1/24. Configure Gateway 182.18.0.1

// Solution:
// Run the command: docker network create --driver bridge --subnet 182.18.0.1/24 --gateway 182.18.0.1 wp-mysql-network
// Inspect the created network by docker network inspect wp-mysql-network
